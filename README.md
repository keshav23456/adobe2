## Canvas - Living Documents That Notice, Remember, and Clarify

Adobe Express add-on that turns documents into living Canvas with awareness features.

### Features

✅ **Publish as Canvas** - Share documents as living Canvas with tracking  
✅ **Blind Spots** - Automatically detect which sections were skipped  
✅ **Milestones** - Save meaningful versions with context ("Client feedback version")  
✅ **Insight View** - Convert paragraphs to bullet points with one click

### Quick Start

**Terminal 1 - Backend:**
```bash
cd backend
npm start
# Backend runs on http://localhost:3000
```

**Terminal 2 - Add-on (already running):**
```bash
npm run start
# Add-on runs on https://localhost:5241
```

### Usage

1. Create content in Adobe Express
2. Open Canvas add-on panel
3. Click **"Publish as Canvas"**
4. Copy the generated link (e.g., `http://localhost:3000/canvas/abc123`)
5. Open link in browser to view Canvas
6. Click **"Create Milestone"** to save versions
7. Click **"View Insights"** to open Canvas viewer
8. In viewer: Toggle **Text | Insights** on any section

### Features in Action

**Canvas Viewer:**
- Real-time tracking (scroll-based)
- Blind Spots panel (right side)
- Milestones history
- Text/Insights toggle per section

**Add-on Panel:**
- Publish as Canvas → Get shareable link
- Create Milestone → Name + reason prompt
- View Insights → Opens Canvas viewer

### Tech Stack

- Adobe Express Add-on SDK
- Node.js + Express
- JSON file-based database
- Tailwind CSS
- Intersection Observer API (tracking)
